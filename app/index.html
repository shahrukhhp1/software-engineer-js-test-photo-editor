Simple Example

Tools

These tools allow you to move and resize image inside canvas, by mouse for better user experience


---------------------------------------------------------

These buttons gives you control to move and resize image inside canvas , you can
change value inside textbox to change difference in moving and resizing

This value can be increased or decreased for change to be apply by following buttons


Save/Import

Submit will save current image and its properties in local storage ,
and import will load that saved info and update canvas with stored info


---------------------------------------------------------

Download button will keep current image and its properties on a file and download it,
Import Config button will allow you to open up that file from file system
on loading that Json File it will load image accordingly as saved in Json File
